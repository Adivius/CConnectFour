# CConnectFour
Multiplayer connect four in C <br>
Player one will be **Host & Client** and player two will just be a **Client**

## Dependencies
> - GCC C17
> - SDL2
> - SDL2 Image 
> - SDL2 TTF 

## Build
Clone project and run
```shell
./build
```

## Run
To start the server run
```shell
./cconnectfour <port> <ip>
```
To join existing game run
```shell
./cconnectfour <port>
```
